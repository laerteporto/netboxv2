Documentação para futuras instalações

Sempre que instalar ou atualizar plugins no NetBox Docker:


docker exec -it netbox bash

cd /opt/netbox/netbox

python3 manage.py collectstatic --noinput

python3 manage.py migrate

exit

docker compose restart

ou:

docker restart netbox

EOF NO FINAL DO SCRIPT

docker cp \
~/netbox-scripts/import_device_images.py \
netbox-docker-netbox-1:/opt/netbox/scripts/
